Program Name: bin.cpp
Author: George Lenz
Date: 8/16/2018

Decription: runs to bin sorting algorithms using the first fit algorithm.
The first runs it on an unsorted list of numbers and the second runs it after sorting the list in decreasing order.

Files included: bin.cpp and mergeSort.hpp

How to run: g++ bin.cpp 

**Must be run on a file named bin.txt, bin.txt file is not included in these files.
bin.txt file must be in the following format:

*******************************************
* Number of test cases
* number of bins
* number of items
* weight of each item with a space betweeen each one
* Repeat second through fourth line for each successive test case
*********************************************
